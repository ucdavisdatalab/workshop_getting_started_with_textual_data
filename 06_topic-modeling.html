6.1.2. Document-term matrix¶

We’ll also quickly make a document-term matrix with scikit-learn’s CountVectorizer(). But since we don’t need to do any preprocessing on our files, we’ll just have CountVectorizer() stream in the files itself using its input argument. All we need to do is give it the list of filenames from our manifest and it will take care of the rest.

from sklearn.feature_extraction.text import CountVectorizer

indir = "data/session_three/input/"
file_names = (indir + manifest['FILE_NAME']).tolist()

vectorizer = CountVectorizer(input = 'filename')
vectorized = vectorizer.fit_transform(file_names)

dtm = pd.DataFrame(
    vectorized.toarray(),
    index = manifest['TITLE'],
    columns = vectorizer.get_feature_names_out()
)
